# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

In short: be kind, be constructive, assume good intent, and help make this a welcoming project for contributors of any background or experience level.

## Enforcement

Reports of conduct violations can be sent to **tejogol@gmail.com**. Reports will be reviewed confidentially and acted on as needed — this may include warnings, edits or removal of contributions, or bans from the project.

## Scope

This applies in all project spaces — issues, pull requests, discussions, and any direct communication tied to contributing to IronClaw.
